# AI Career Coach

## Problem
Many data analyst freshers in India fail to get shortlisted due to poor resume alignment with job requirements.

## Solution
This project analyzes resumes against real job market data and provides:
- Market match score
- ATS compatibility
- Skill gap analysis
- Hiring probability insights

## Tech Stack
- Python
- PostgreSQL
- NLP
- FastAPI (upcoming)
- Streamlit (upcoming)
